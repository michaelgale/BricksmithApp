Keyboard Shortcuts

Bricksmith provides many "secret" keyboard shortcuts to help building go faster.

Note: Many of Bricksmith's menu items also have shortcuts printed next to them. (For instance, Grid Spacing > Small is "⌘1".) This document only covers keys that you can't find elsewhere.

* These are known to conflict with Mac OS X's default keys for displaying the Spotlight window. They were chosen to maintain compatibility with Adobe's application suite, which has been using these keys for zoom in/out for well over a decade.

To compensate, I (and Adobe) recommend one of the following:

  • Press the spacebar first, then press the command key.
  • Turn off or change the keys associated with Spotlight.
    You can do this by opening the System Preferences and going to the Keyboard & Mouse panel. Click on the "Keyboard Shortcuts" tab, and scroll down to the Spotlight category. Then either uncheck or change them.
